#### 01  请解释为何下边变量 a 的值为何打印出来是 144（请写出分析过程）？

```c
#include <stdio.h>

int main()
{
        unsigned char a, b, c;

        b = c = 200;
        a = b + c;

        printf("a = %u\n", a);

        return 0;
}
/*
无符号char类型所能表示的最大数字是255
变量a此时的值为：400
400 - 255 = 145
超出的数字也要从0开始
因此最后的结果输出就是144
*/
```
